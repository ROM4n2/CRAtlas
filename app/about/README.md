# app/about/

关于页——项目说明、意识形态立场声明、史料来源说明、使用指南。

## 职责

关于页是静态内容页面，无需 `'use client'`。向用户阐明项目宗旨、编纂立场、史料来源类型和基本使用方式，是用户了解项目背景的入口。

## 文件清单

| 文件 | 职责 |
|------|------|
| `page.tsx` | 关于页——四大内容区块（项目说明、立场声明、史料来源、使用指南） |
| `README.md` | 本文件 |

## 使用方式

关于页为独立路由页面，通过顶部导航栏"关于"链接访问：

```tsx
// 导航至 /about 即可渲染本页面
<Link href="/about">关于</Link>
```

页面为服务端组件（Server Component），无需客户端运行时。

## 约定

- **纯静态内容**：不包含任何交互逻辑、状态管理或数据查询
- **内容对齐**：内容定义参照设计规范 §5.5 关于页内容定义
- **样式**：使用 Tailwind CSS 工具类，与全站风格保持一致
- **布局**：`max-w-3xl mx-auto` 居中阅读宽度，适合长文阅读
