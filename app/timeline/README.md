# app/timeline/

时间轴视图页——横向时间轴 + 事件卡片流，按时间顺序展示 1966-1976 年间派系相关事件。

## 职责

时间轴视图以纯列表形式呈现历史事件，配合底部 TimeAxis 控制条，让用户通过拖动时间轴浏览任意时间点之前的所有事件。与地图视图共享同一时间状态，三视图（地图/图谱/时间轴）时间同步。

## 文件清单

| 文件 | 职责 |
|------|------|
| `page.tsx` | 时间轴页面——订阅 useTimeStore，调用 getEventsByDate 获取事件列表，渲染 EventCard 列表 + 底部 TimeAxis |
| `README.md` | 本文件 |


## 使用方式

时间轴视图为独立路由页面，通过顶部导航栏"时间轴"链接访问：

```tsx
// 导航至 /timeline 即可渲染本页面
<Link href="/timeline">时间轴</Link>
```

页面为 `'use client'` 组件，必须放在客户端渲染上下文中。

## 约定

- **时间同步**：当前日期通过 `useTimeStore` 全局共享，与地图、图谱视图保持同步
- **布局**：采用 `flex flex-col h-[calc(100vh-3.5rem)]` 全屏布局，时间轴固定在底部
- **空状态**：当 `getEventsByDate` 返回空数组时，显示"当前时间点暂无事件"
- **事件卡片**：通过 `components/timeline/EventCard.tsx` 展示，包含标题、日期、描述与详情链接
- **重要性标识**：事件卡片右上角显示 importance 标签（重大/重要/一般）
